# ansidev's Homebrew Casks

## Install

```sh
brew install --cask ansidev/casks/{cask}
```

Or

```sh
brew tap ansidev/casks && brew install --cask {cask}
```

## Available Casks

- mac-updater.
- peazip.
- simplenote.

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
